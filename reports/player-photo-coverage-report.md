## How to Add a Player Photo

1. Add image file to `public/player-photos/<nickname>.png`
2. Add entry in `src/data/playerPhotoOverrides.ts`:
   ```ts
   "Nickname": { src: "/player-photos/nickname.png", scale: 1.3, offsetY: 12 },
   ```
3. Run `npm run report:player-photos` to update this report.

## How to Add Approved Photo Sources

1. Edit `data/player-photo-sources.json` with approved URLs
2. Run `npm run sync:player-photos` to download
3. Do NOT add unapproved/copyrighted images

## Paths

- Photo folder: `public/player-photos/`
- Override map: `src/data/playerPhotoOverrides.ts`
- Sources manifest: `data/player-photo-sources.json`
